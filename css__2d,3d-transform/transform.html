CSS Transform

CSS Transform 2D

translate (x, y)

Տեղափոխում է elemente դեպի նոր տեղ՝ այն տեղափոխելով աջ և ներքև՝ օբյեկտի սկզբնական դիրքի համեմատ՝ օգտագործելով X և Y կոորդինատները:

scale (x, y)

մասշտաբում է տարրերը՝ դրանք մեծացնելով կամ փոքրացնելով: Լռելյայնորեն, արժեքները նշված են որպես միավոր (1) - սա օբյեկտի իրական չափն է: Առաջին արժեքը չափում է տարրը լայնությամբ, երկրորդը՝ բարձրությամբ: Բացասական արժեքները արտացոլում են տարրը:

rotate (անկյուն+deg)

պտտել - պտտում է տարրերը նշված աստիճանների քանակով, բացասական արժեքները պտտում են տարրը ժամացույցի սլաքի ուղղությամբ, դրական արժեքները պտտում են տարրը ժամացույցի սլաքի ուղղությամբ: Արժեքները գրվում են որպես ռոտացիայի աստիճաններ և աստիճաններ

skew (x-անկյուն + deg, y-անկյուն + deg)

դեֆորմացնում (խեղաթյուրում է) օբյեկտի կողմերը կոորդինատային առանցքների համեմատ: Եթե ​​նշված է մեկ արժեք, ապա երկրորդը կորոշվի ավտոմատ կերպով բրաուզերի կողմից: Արժեքները գրվում են որպես ռոտացիայի աստիճաններ և աստիճաններ

Բազմաթիվ 2D փոխակերպումներ
transform:
translate(0, 0)
scale(0.7, 0.7)
rotate( 90deg)
skew(0deg, 0deg);

Բացի այդ, դրան կարելի է հասնել բազմաթիվ փոխակերպումների միջոցով: Մասնավորապես, գրեք բազմաթիվ արժեքներ մեկ փոխակերպման հատկության մեջ:

transform-origin

transform-origin- թույլ է տալիս տեղափոխել վերափոխման կենտրոնը, որի համեմատ փոխվում է elementi դիրքը / չափը / ձևը

CSS Transform 3D

perspective, perspective-origin

perspective, perspective-origin- խորությունը սահմանելը և ծագման կետը փոխելը

translate3d (x, y, z)

translate3D - ի անալոգիա իր 2D եղբոր հետ, այս արժեքը սահմանում է տարրի շարժումը, բայց 3D տարածության մեջ: Բացի X և Y առանցքներից, ավելացվում է Z առանցքը:

scale3d (x, y, z)

scale3D - սահմանում է եռաչափ mashtab X, Y, Z առանցքներից մեկի երկայնքով

rotate3d (x, y, z, deg)

rotate3d - պտտում է elemente երեք առանցքի շուրջ: elemente պտտվում է վերջին պարամետրով սահմանված անկյան տակ X, Y, Z ուղղության վեկտորի նկատմամբ

Բազմաթիվ 3D փոխակերպումներ

transform: translate3d(0, 0, 0) scale(0.7, 0.7, 0.7) rotate(1, 1, 1, 90deg)

Բազմաթիվ 3D արժեքներ գրելը մեկ փոխակերպման հատկության մեջ:

transform-style

transform-style- սահմանում է փոխակերպման ոճը

Backface-visibility
