overflow

overflow- այս հատկությունը ազդում է «լրացուցիչ» բովանդակության ցուցադրման վրա, որը դուրս է գալիս բլոկի սահմաններից:

  • visible - default
  • hidden - թաքցնում է ավելորդ բովանդակությունը
  • scroll - ավելացնում է 2 scrollbar անկախ դրանց կարիքից
  • auto - անհրաժեշտության դեպքում ավելացնում է scrollbar
