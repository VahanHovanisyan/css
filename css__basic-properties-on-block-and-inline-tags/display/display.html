display

display- ազդում է էլեմենտի բուն էության վրա:

  • block - բլոկ էլեմենտ զբաղեցնում է ամբողջ տօղի տարածքը
  • inline - տօղային էլեմենտ
  • inline-block - բլոկի և տօղայինի հիբրիդն է, բայց ավելի քիչ «ագահը», քանի որ այն չի զբաղեցնում ամբողջ տարածքը, ինչպես բլոկը:
  • none - ջնջումա էլեմենտ ը
  • flex grid ... - կանցնենք միքիչ ուշ
